# Which system should be used for the application?

### Environment: Webapp

We are going with a webapp environment. This will allow us to quickly set up a cross-platform application that can display data and inform users of their progress in real time.

The reason for choosing a webapp over a mobile application is primarily due to time constraints and limited development resources. Developing a mobile app would require additional effort for platform-specific compatibility, deployment, and testing across both Android and iOS, which is not feasible within our current timeline.

By focusing on a web-based solution, we can streamline the development process and allocate more of our resources toward implementing machine learning models for activity detection—one of the core technical challenges of this project. A webapp also provides easier access for users without requiring installation, making it ideal for all kinds of devices and users.

Overall, the webapp environment aligns best with our project goals, available skills, and time constraints.
