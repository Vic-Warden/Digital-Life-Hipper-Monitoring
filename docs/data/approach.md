# Plan of Approach for the Next Round of Tests

## 📋 Step-by-Step Plan

### Test Setup

* Securely fix sensors in housings to prevent mechanical variance.

* Tape two units together for direct comparison.

* Use sensors on healthy subjects and revalidanten if possible.

### Data Collection

* Target duration: Sessions of 10 minutes as discussed with Michel.

* Log both raw sensor data and minute score outputs.

### Analysis Metrics

* Compare PAM values:

  * Direct from sensor

  * Summed from minute data

* Analyze:

  * Minute-level score patterns

  * SSE between sensor outputs

  * Differences in activity classification

### Activity Classification

* Apply both:

  * Erik’s thresholds (for low/medium/high activity)

  * Simplified “Active vs. Inactive” threshold (e.g., 0.002–0.4 PAM/min)

* Examine impact of different thresholds on consistency

### Optional

Consider testing more than two sensors simultaneously to assess inter-sensor variance across a larger sample.
