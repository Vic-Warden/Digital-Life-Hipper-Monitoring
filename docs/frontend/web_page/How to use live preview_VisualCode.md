# How to Use Live Preview in Visual Studio Code

This is a simple guide for previewing your HTML files in Visual Studio Code using the **Live Preview** extension.

---

## Steps

### 1. Open Visual Studio Code
Start VS Code on your computer.

### 2. Install Live Preview
- Click the Extensions icon on the left sidebar.
- Search for `Live Preview`.
- Click **Install** on the one by Microsoft.

### 3. Open an HTML File
- Open a folder or create a new file with `.html` extension.
- Example: `index.html`

### 4. Start the Preview
- Click the **Go Live** button in the bottom-right corner.
- Or press `Ctrl+Shift+P` and type `Live Preview: Show Preview`.
