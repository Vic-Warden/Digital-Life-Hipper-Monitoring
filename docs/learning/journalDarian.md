#### Learned:
by doing the other user story about how to implement powerbi in a web application. I learned that it is possible to make an iframe which can be used in HTML to showcase the powerbi. 

the steps to take are found and are as follows:
### 1. Create and Publish a Power BI Report
- Use **Power BI Desktop** to create a report using sample/test data.
- Click `File > Publish > Publish to Power BI`.
- Select `My Workspace`.

### 2. Generate Embed Code
- Go to [Power BI Service](https://app.powerbi.com)
- Open your report in the workspace.
- Click `File > Embed report > Publish to web (public)`
- Confirm and copy the iframe code or public URL.
